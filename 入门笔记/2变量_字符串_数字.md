# 字符串
修改字符串大小写
```python
name = "ada"
print(name.title()) #首字母大写
print(name.upper()) #全部大写
print(name.lower()) #全部小写
```
拼接
```python
first_name = "ada"
last_name = "lovelace"
full_name = first_name + " " + last_name
print("Hello, " + full_name.title() + "!")
```
\t,制表符 \n换行, 也可以\n\t换行对齐
删除空白
```python
variable.rstrip() #去除末尾空白
variable.lstrip() #去除开头空白
varibale.strip()  #去除两端空白
```
